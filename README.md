# Parser

## Requirements

- PHP
- Composer
- Redis server

Install composer requires

    composer install

Create & change your .env file

    cd src
    cp .env.example .env

## Run script

This command is used to run script from scratch

    php scrap.php init

If you need to restart script please use this command

    php scrap.php

Check the relevance of ads

    php scrap.php check
